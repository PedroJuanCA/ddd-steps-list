# ddd-steps-list

A custom PSU HAX + DDD web component for Project 1.

## What it does

- `<ddd-steps-list>` wraps `<ddd-steps-list-item>`
- Validates children (only item tags allowed)
- Auto-generates step numbers
- Passes down `ddd-primary` for color
- Styled to match the PSU 2+2 Plan layout
